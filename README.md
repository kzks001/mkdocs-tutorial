# MkDocs Tutorial

## Getting Started
### Prerequisites
Ensure you have pdm installed. You can install it using `pipx`:

```sh
pip install --user pipx
pipx ensurepath
pipx install pdm
```
### Installation
Within the root of your project, run:

```sh
pdm install
```
### Running MkDocs
To see the output of MkDocs, use:

```sh
mkdocs serve
```
